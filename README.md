# 珍珠账本 · Pearl Ledger
> SillyTavern 扩展

---

## 安装方法

### 第一步 — 上传到 GitHub

1. 登录你的 GitHub 账号
2. 新建一个仓库，名字随意，比如 `pearl-ledger`，设为 **Public**
3. 把这个文件夹里的三个文件全部上传进去：
   - `manifest.json`
   - `index.js`
   - `style.css`
4. 上传完成后，记下你的仓库地址，格式是：
   `https://github.com/你的用户名/pearl-ledger`

---

### 第二步 — 在酒馆安装扩展

1. 打开你的云酒馆
2. 点击顶部的 **扩展（Extensions）** 图标（拼图形状）
3. 找到 **Install Extension** 输入框
4. 粘贴你的 GitHub 仓库地址，点击安装
5. 刷新页面

---

### 第三步 — 使用

- 安装后，**扩展面板**里会出现「珍珠账本 · Open」按钮
- 点击打开弹窗，选择收入/支出，输入金额和备注，点击「记录」
- 数据保存在浏览器本地，不会因为刷新消失
- 点击「📤 发给爸爸看」会把账单摘要自动发送到对话里

---

## 功能说明

| 功能 | 说明 |
|------|------|
| 收入 / 支出切换 | 顶部两个按钮切换类型 |
| 金额 | 支持小数 |
| 备注 | 最多 50 字 |
| 删除记录 | 每条记录右侧 × 按钮 |
| 发给对话 | 整理近 5 条记录 + 总计发送 |
| 持久保存 | localStorage，换设备需重新同步 |
